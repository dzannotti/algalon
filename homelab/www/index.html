Connect to Algalon

To access all services via *.algalon.local, configure your device to use Algalon as its DNS server.

1 Open Settings → Network & Internet → Wi-Fi or Ethernet
2 Click your connection, then Edit next to DNS server assignment
3 Switch to Manual, enable IPv4, set DNS to: 192.168.1.100
4 Save and visit algalon.local to verify
1 Open System Settings → Network
2 Select your connection (Wi-Fi/Ethernet), click Details
3 Go to DNS, click + and add: 192.168.1.100
4 Click OK and visit algalon.local to verify
1 Open network settings or run: nmcli con mod "CONNECTION" ipv4.dns "192.168.1.100"
2 Restart connection: nmcli con down "CONNECTION" && nmcli con up "CONNECTION"
3 Or edit /etc/resolv.conf and add: nameserver 192.168.1.100
1 Log into your router admin panel (usually 192.168.1.1)
2 Find DHCP Settings or LAN Settings
3 Set Primary DNS to: 192.168.1.100
4 Save and reboot router. All devices will use Algalon DNS automatically!

Access Shared Folder

1 Open File Explorer and click This PC
2 Click Map network drive in the top menu
3 Pick a drive letter and enter: \\192.168.1.100\data
4 Check "Reconnect at sign-in" and click Finish
